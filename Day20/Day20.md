# 🚀 Day 20 – AI Face Puzzle Game with Claude AI

## 📌 Overview

On **Day 20** of the **60 Days Claude AI Challenge**, I explored how Claude AI can generate a complete interactive web application from a single prompt.

Today's challenge focused on building a **Face Puzzle Game**, where users can capture their own photo using the webcam and instantly convert it into an interactive sliding puzzle with multiple difficulty levels.

The application was developed as a fully functional HTML project using HTML, CSS, and JavaScript, demonstrating AI's ability to generate creative and engaging browser-based applications.

---

# 🎯 Objective

- Build an AI-generated Face Puzzle Game.
- Capture images using the webcam.
- Create personalized puzzle pieces.
- Implement interactive drag-and-drop gameplay.
- Track player performance using timers and move counters.
