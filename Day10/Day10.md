# 🚀 Day 10 – Building My Personal Portfolio Website with Claude AI

## 📌 Overview

On **Day 10** of the **60 Days Claude AI Challenge**, I explored how AI can be used to design and generate a complete **personal portfolio website** from a single prompt.

Using the provided Portfolio Website prompt, I customized the information with my personal details, skills, projects, and career goals. Claude then generated a complete HTML portfolio website that showcases my profile as an aspiring **AI Engineer and Software Developer**.

This challenge demonstrated how AI can significantly accelerate web development and personal branding.
